## ✨ Features

### 🔐 Core Visitor Management
- **Digital Check-In/Check-Out** - Streamlined registration with photo capture
- **Pre-Registration System** - Schedule visits and send QR code invitations
- **Real-Time Visitor Tracking** - Live status updates and location monitoring
- **Digital Badge Printing** - Customizable visitor badges with QR codes
- **Host Notifications** - Automatic alerts to employees when visitors arrive

### 📱 Advanced Technologies
- **QR Code Scanning** - Contactless, fast-track check-in support
- **Photo Capture** - ID verification with camera integration
- **Responsive Design** - Works seamlessly on desktop, tablet, and mobile
- **Glassmorphism UI** - Modern, visually stunning interface design
- **Dark Mode** - Eye-friendly dark theme with vibrant accents

### 🚨 Safety & Security
- **Emergency Evacuation Mode** - One-click mustering and roll-call system
- **Real-Time Accountability** - Track who is safe vs. missing during emergencies
- **Watchlist Screening** - Security checks against banned visitor lists
- **Audit Trail** - Complete logs of all visitor activities
- **Data Export** - CSV export for compliance and reporting

### 📊 Analytics & Reporting
- **Visitor Statistics** - Daily, weekly, and monthly traffic insights
- **Peak Hour Analysis** - Identify busy periods for resource planning
- **Department Tracking** - Monitor visits by host department
- **Custom Reports** - Export data for management review
